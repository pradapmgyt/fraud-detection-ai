💳 AI Fraud Detection System
🚀 Project Overview

This project is a Machine Learning-based fraud detection system that identifies whether a transaction is fraudulent or safe using real transaction data.

🧠 Features
Detects fraud using Random Forest model
Handles imbalanced data using SMOTE
Real-time prediction system
Streamlit web interface
⚙️ Tech Stack
Python
Pandas, NumPy
Scikit-learn
Imbalanced-learn
Streamlit
🖥️ How to Run
pip install -r requirements.txt
streamlit run app.py
📌 Note

Dataset is not included. Download from Kaggle:
https://www.kaggle.com/datasets/mlg-ulb/creditcardfraud

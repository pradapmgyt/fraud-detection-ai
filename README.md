# 💳 AI Fraud Detection System

🚀 A machine learning project that detects fraudulent transactions using Random Forest and SMOTE, with a real-time Streamlit web interface.

---

## 🔥 Highlights

* ⚡ Real-time fraud prediction
* 🧠 Handles imbalanced datasets using SMOTE
* 🎯 High precision detection
* 🌐 Interactive web app using Streamlit

## 🧠 Tech Stack

* Python
* Scikit-learn
* Pandas, NumPy
* Imbalanced-learn (SMOTE)
* Streamlit

---

## 📊 Model Performance

* Accuracy: ~99%
* Fraud Recall: ~83%
* Balanced dataset using SMOTE

---

## ⚙️ How it works

1. User enters transaction details
2. Model processes input features
3. Random Forest predicts fraud probability
4. Result displayed instantly

---

## 🚀 How to Run

```bash
git clone https://github.com/pradapmgyt/fraud-detection-ai.git
cd fraud-detection-ai

pip install -r requirements.txt
streamlit run app.py
```
